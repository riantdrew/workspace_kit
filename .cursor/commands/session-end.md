# session-end

Принудительная консолидация памяти сессии.

1. `python3 .cursor/hooks/session_memory.py session_end`
2. `python3 .cursor/hooks/session_memory.py consolidate_orphans`
3. Проверить `workspace/riantdrew/memory/YYYY-MM-DD.md`
4. При необходимости сжать важное в `workspace/MEMORY.md` (без секретов)

Тот же эффект, что фраза «завершаем сессию» в BOOTSTRAP.
